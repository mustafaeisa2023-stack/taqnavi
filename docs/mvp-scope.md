# MVP Scope — Taqnavi

## Must-Have Features
- Start screen with clear CTA to begin quiz
- Scenario-based quiz (single-select answers)
- Deterministic weighted scoring across six tracks
- Result screen with top match and top 3 ranked tracks
- Short explanation for why the top match appeared
- Restart flow to begin a new attempt quickly
- Mobile-first, booth-friendly layout and readable UI

## Should-Have Features
- Progress indicator (e.g., question count)
- Lightweight transition between steps for clarity
- Concise “next steps” suggestions per track
- Basic validation to prevent empty submissions

## Could-Have Features
- Optional review of previous answers before submit
- Printable/shareable summary view (no account required)
- Theme polish for event branding

## Won't-Have Features for MVP
- Backend services
- Database persistence
- Login/authentication
- AI API integration
- Admin dashboard
- Firebase/Supabase integration

## Out of Scope
- Scientific personality diagnosis claims
- Long-form psychometric testing
- Personalized mentorship matching
- Job placement or certification guarantees
- Multi-language localization in MVP

## MVP Success Criteria
- Users can complete the quiz and view results in under 3 minutes
- Every submission returns a deterministic top match and top 3 list
- Result language clearly states guidance (not diagnosis)
- App supports quick restart for repeated booth usage
