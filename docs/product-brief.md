# Product Brief — Taqnavi

## Product Name
Taqnavi

## One-Sentence Description
Taqnavi is a booth-friendly guided recommendation web app that asks scenario-based questions and suggests tech career tracks based on user answers.

## Target Users
- High school and university students exploring tech careers
- Career fair and campus booth visitors needing a fast recommendation
- Beginners who want a clear first direction in technology

## Problem Statement
Students interested in technology often do not know where to start, and generic advice is usually too broad, time-consuming, or hard to compare during short interactions like events or booths.

## Solution Summary
Taqnavi provides a short, structured quiz with weighted scoring across six predefined tech tracks. At the end, users receive a top recommendation plus alternatives and a practical next-step summary.

## Value Proposition
- Fast and simple: complete in a few minutes
- Structured guidance: compares multiple tracks consistently
- Action-oriented: provides clear “what to do next” direction
- Booth-ready: minimal friction and restart-friendly flow

## Disclaimer
Taqnavi recommendations are generated from user-provided answers using a transparent weighted scoring model. They are guidance only and **not** a scientific diagnosis, personality test, or psychometrically validated assessment.
